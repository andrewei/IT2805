3. Purpose, goals and audience

Purpose and Goals

The goal of the website is for anyone and everyone interested in the Smash game series to not only get information about the Smash Trondheim community but also the games, both general and competitively. It will also serve as a place where people interested in the game or the organisation can go to get information or get pointers to where they could go next to learn more.
Almost all pages will include external links or bring in information from other websites. It will also feature a page which consists of links to different pages and sites that Smash Trondheim is either a part of or involved with.
Furthermore, there will also be a page listing recent and upcoming events with concise information and relevant external links. The website should have the same style on each page and it should be easy to navigate.

Audience

The main demographic are the people most likely interested in the game series; people with an existing interest and people who wish to learn more. The majority of our demographic consists of young, male adults. This is the demographic which most often turn up for events and also are the most active participants in the group as a whole.
Nonetheless, this site should be made with every interested party in mind. This entails that the main demographic may not become a factor which would change the design into something which would create an uncomfortable feeling for anyone new and interested. Therefore, we want to keep the website simple and easy to use for everyone. Gaming websites have a tendency to run darker colors on their websites, and we have consciously decided to choose a similar color palette.

4. Navigation Structure

All pages link to all main pages on the website. In each main page/category on the website, you will find links to all the sub-pages under that respective page/category. The maximum number of clicks required to get from one place on the website to another is two. You can get to the relevant main page from anywhere on the site, and from it is at most one click to get to the desired sub-page.

6. Content

There will be 5 separate main pages with a number of sub-pages as follows:

  1. Main

    This page is the first page the user sees. The main function of this page is to introduce Smash Trondheim and what they do.
    There will also be a youtube-video of the Smash for Wii U intro sequence that is playable.

  2. Events and Results
    1. Current Events(default mainpage)
    2. Previous Events

    Both pages will have a link to the other page, so there is a possibility of moving back and forth.
    Both pages will use the same table type, where the first row is the name of the event with a link to the Facebook event. Third row will be the date of the event. The last, fourth, row will be a navigation bar with a clickable picture linking to the respective tournament bracket.
    In Current Events, there should be information on S@NG, the weekly event, and information about both when and when it is not happening. In addition, there should not be more than 5 events in current events at any given time, with past and future combined.
    Previous events is a list of all events no longer in current events. The list is going to be reset every new semester.

  3. Games
    1. Mainpage(default mainpage)
    2. Melee
    3. PM
    4. Brawl
    5. Smash for Wii U

    The content on the main page is mainly an explanation of why we play the games that we do and shortly about the other games of the franchise are not being played.
    For every game-specific subpage page, there will be information about the game, the game competitively, activity in Smash Trondheim and also a Youtube-video of the intro for the game (PM should use latest release video). Links to Wikipedia or other informational sites will also be present.1

  4. Links

    This page will be a collection of relevant links to external resources. These links may take you to a community for a specific game or even the series in general. There are several sites which may be put here (e.g. Facebook, Twitch, Challonge, Youtube) and there may even be multiple pages from each of these sites.

  5. Contact

    This page will contain a list of the board members of Smash Trondheim with names and e-mail addresses. The intention with this page is to provide contact information to the board in case of questions. Additionally, if time allows, we wish to create a form where visitors may insert messages that will be sent to members of the board.

7. Minimum Requirements

We plan to fulfil the minimum requirements of Javascript applications in this way:

  1. Make a Javascript that dynamicly changes the contents of the Game page to display information about a selected game. It will also light up the icon on the selected game.
  2. Make certain buttons play an animation clip after key press.
  3. Create a javascript to load in different elements of webpages that is used different places. For instance the navbar and footer. Templating.
  4. Javascript code to display a box in the upper right corner with event information in the case of a currently running event.
